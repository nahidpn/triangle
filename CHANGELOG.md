# Changelog

All notable changes to TRIANGLE project are documented in this file.

## [1.0.0] - 2026-06-30

### Added
- Initial release of TRIANGLE deepfake detection platform
- FastAPI backend with image and video detection endpoints
- CNN + MTCNN model implementation
- React.js web UI for interactive analysis
- Docker and docker-compose configuration
- Comprehensive API documentation
- GitHub Actions CI/CD pipeline
  - Automated testing (pytest)
  - Code quality checks (flake8, mypy)
  - Security scanning (bandit, CodeQL)
  - Docker build caching
- MIT License
- Contributing guidelines
- Project documentation (README)
- Performance benchmarks (92% accuracy, <500ms inference)

### Features
- Image deepfake detection endpoint
- Video deepfake detection with frame-by-frame analysis
- Health check endpoint
- CORS support for cross-origin requests
- Async processing for better performance
- Error handling and validation

### Documentation
- API endpoint documentation
- Setup and installation guide
- Docker deployment instructions
- Azure cloud deployment guide
- Model architecture diagrams
- Performance benchmarks
- Troubleshooting section
